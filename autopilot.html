01 — Motivation

The Gap Between
Autonomy & Agency

Modern autonomous vehicles excel at routine tasks — but falter when human judgment is needed. We bridge that gap.

🚒
Rigid Decision-Making
In 2023, a Waymo robotaxi blocked a responding fire truck by adhering rigidly to safety rules without reading the broader situation.
🎛️
Clunky Override Interfaces
Passengers who want to adjust speed, change lanes, or reroute must use complex physical controls — disrupting the pipeline.
🤝
The Shared Autonomy Gap
Systems force a binary: fully manual or fully autonomous. Real use demands a continuum where humans guide and machines execute.
⚡
Our Solution
Natural multimodal input — hands and voice — mapped to vehicle commands in real time, with safety constraints as guardrails.
02 — Architecture

Two Subsystems,
One Control Loop

Autonomous Vehicle

Planning
A* Route Planner
Graph-based path search over junction nodes. Cost = road segment length + Euclidean heuristic.
Navigation
Waypoint Tracker
Interpolates lane-centered waypoints along straight and curved segments with pure-pursuit steering.
Perception
Lane Detection
OpenCV HSV thresholding on yellow/white markings + column histogram for lane position.
Perception
Obstacle Detection
MediaPipe EfficientDet; bounding-box vertical position as geometric depth proxy.
SHARED
CONTROL
LOOP

Human Interaction

Input
Gesture Recognizer
MediaPipe hand landmarks on webcam. ✊ stop · 👍 kerb lane · 👎 inner lane. Background thread.
Input
Speech Commands
"faster" / "slower" → ±1 m/s. "left" / "right" near junctions → turn override + A* reroute.
Safety
Priority & Constraints
Gesture stop → obstacle avoidance → human lane change → autonomous nav. Lane boundary checks gate unsafe commands.
Fallback
Safe Stop + Handoff
Both lanes blocked → vehicle halts and waits for human input to resume.

06 — Results

System Performance

<100ms

Interaction Modalities (click, gesture, speech)
2-lane
Obstacle Avoidance with Safe Handoff Fallback

The system demonstrated reliable shared-autonomy behavior across test scenarios. Human commands integrated without requiring full takeover. Obstacle avoidance triggered on single-lane blockages; safe halt when both blocked. Speech turn overrides rerouted within the 6-second junction window, and gesture lane changes fired with a 50-step avoidance suppression cooldown.
